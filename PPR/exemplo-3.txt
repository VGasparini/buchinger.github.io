Gostaríamos de fazer o pedido de elaboração do seguinte sistema:

Estamos com um problema no gerenciamento das vagas de estacionamento da universidade. Gostaríamos que o sistema fosse capaz de registrar a entrada dos veículos na universidade e informar ao motorista qual vaga estará disponível para ele.
Na saída, o sistema deve registrar a saída do carro e desalocar a sua vaga. A resposta do sistema deve responder em no máximo 5 segundos. Se não há vagas disponíveis, o sistema solicita para que o motorista espere ou dê a volta. Devemos destacar ainda que o sistema deve manter a área de funcionários restrita para o estacionamento de carros de funcionarios.

Já possuímos um cadastro de placas de automóveis que são permitidos na universidade, mas o sistema deve permitir novas inclusões de automóveis de: alunos, professores, funcionários e carros oficiais. Neste mesmo sentido ainda, somente o funcionário adequado poderia fazer as inclusões de placas.
Gostaríamos que o sistema pudesse ser utilizado no windows 7 e no ubuntu 10.04.




Requisitos Funcionais:
- O sistema deve reistrar a entrada e saída de automóveis na universidade
- O sistema deve informar ao motorista qual vaga está disponível para ele
- O sistema deve gerenciar as vagas de estacionamento, alocando vagas desocupadas para novos automóveis e liberando as vagas de automóveis que deixarem o espaço
(...)


Requisitos Não-Funcionais:
- O sistema deve ser suportado para os sistemas operacionais Windows (Windows 7) e Linux (Ubuntu 10.04)
- O sistema deve responder em até 5 segundos a uma solicitação de alocação de nova vaga de estacionamento
(...)


Requisitos Normativos
- O sistema deve reservar uma área do estacionamento exclusiva para funcionários
(...)
